# Longhorns Racing: The Game

A 2D top-down browser racing game built with HTML5 Canvas and JavaScript.

## Setup

```
npm install
npm start
```

Open http://localhost:3000 in your browser.

## Controls

| Key | Action |
|-----|--------|
| Arrow Up / W | Accelerate |
| Arrow Down / S | Brake / Reverse |
| Arrow Left / A | Steer left |
| Arrow Right / D | Steer right |
| Space | Handbrake (drift) |
| Escape | Pause |

## Tracks

1. **Darrell K Royal** — Oval with chicanes
2. **Sixth Street Circuit** — Technical city circuit
3. **Hill Country Dash** — Figure-eight with crossover

## Features

- 3 unique track layouts
- 4 AI opponents with varying difficulty
- Arcade-style physics with drifting
- Procedural audio (engine, collisions, music)
- HUD with speed, lap times, minimap, standings
- Skid marks and particle effects
